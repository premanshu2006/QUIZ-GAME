# Contributing to Quiz Game

Thanks for your interest in contributing! Follow these steps:

1. Fork the repository.
2. Create a feature branch: `git checkout -b feature/your-feature`.
3. Install dependencies and run the app:
   - `npm install`
   - `npm start`
4. Make code changes in the `public/` UI, `server.js`, or README as needed.
5. Add tests (if applicable).
6. Commit with clear message:
   - `git commit -m "Add ..."`
7. Push to your branch and open a pull request.

## Coding style
- JavaScript: ES6+, use `const` and `let`.
- Keep UI responsive and accessible.
- Use meaningful variable names.

## Reporting issues
- Open an issue and include steps to reproduce, the browser, and Node version.
